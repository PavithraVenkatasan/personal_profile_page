# Personal Profile Page

A simple and responsive personal profile webpage built using **Semantic HTML5**. This project demonstrates the proper use of semantic HTML elements to create a well-structured and accessible web page without using CSS or JavaScript.

## Objective

The purpose of this project is to practice semantic HTML by organizing content into meaningful sections while following standard HTML5 document structure.

## Features

- Semantic HTML5 structure
- Proper HTML boilerplate
- Header section with name and role
- About Me section
- Skills section with an unordered list
- Contact section with email, phone, and LinkedIn link
- Clean and beginner-friendly code
- No CSS or JavaScript used

## Technologies Used

- HTML5

## Semantic HTML Elements Used

- `<!DOCTYPE html>`
- `<html>`
- `<head>`
- `<body>`
- `<header>`
- `<main>`
- `<section>`
- `<h1>` and `<h2>`
- `<p>`
- `<ul>` and `<li>`
- `<a>`

## How to Run the Project

1. Clone this repository.
2. Open the project folder.
3. Open the `index.html` file in your web browser.

Or use the **Live Server** extension in Visual Studio Code for a better development experience.
